# SSLF
SSLF(Stupid Simple Language Fuck) is low level language.
SSLF ​​is similar to assembly but not exactly the same.

syntax : $
rule : '$' only for variables and hexadecimal for instruction to CPU.

example : '$ bite 0xFF'
it representation of 1 bite in CPU using hexadecimal.
im not sure this language is useful for everyone in embedded system.

i have idea for this language to support int number and float number for easier to write.
use ^ for representation of hexadecimal but using int or float number.

example : '$ bite 1^2'
it same like '$ bite 0xFF'.
